Projet d'animation d'un drapeau avec un système de particules

Installation :
    
    Le projet a été écrit en C++ en utilisant la libg3x. Il donc nécessaire d'avoir installer la libg3x et le compilateur g++.

Compilation et exécution :

    cd FLAG_PROJECT
    make flag
    ./flag

Utilisation :

    Appuyer sur "R" pour réinitialiser l'état du drapeau.
    Appuyer sur "ESPACE" pour lancer l'animation et l'arrêter.
    Appuyer sur "S" pour afficher en couleur les vecteurs de force des liens, ou pour revenir à l'affichage des quads.
    Appuyer sur le bouton de la molette + translater la souris pour zoomer.
    Appuyer sur le clic gauche pour effectuer une rotation autour du drapeau.

Détails :

    "FLAG_PROJECT/flag.cpp" est le fichier main du projet.
